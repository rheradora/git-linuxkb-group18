System Troubleshooting



Command: uptime

Description: Displays how long the system has been running, the number of logged-in users, and the current system load.

example: uptime



Command: dmesg

Description: Displays kernel messages that can help identify hardware, driver, and startup issues.

Example: dmesg | tail



Command: journalctl

Description: Displays system logs and helps investigate errors or failed services.

Example: journalctl -p err
